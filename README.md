# employee-performance-dashboard
Excel-based Employee Performance Dashboard

Employee Performance Dashboard (Excel)
Project Overview
This project is an Excel-based Employee Performance Dashboard designed to analyze and visualize employee performance using key performance indicators (KPIs). The dashboard helps in understanding individual and overall performance to support HR decision-making.

Objective
To create an interactive and easy-to-understand dashboard that evaluates employee performance based on:
•Sales Achievement
•Attendance
•Productivity
•Overall Performance Score

Tools Used
Microsoft Excel
Formulas
Charts (Column Charts)
Data Analysis
